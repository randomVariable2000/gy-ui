## API

| 属性 | 说明     | 类型                                         | 默认值 |
| ---- | -------- | -------------------------------------------- | ------ |
| kind | 警告类型 | 'info'/'positive'/'negative'/'warning'非必填 | 'info' |

## 难点
* 聚焦与失焦时对输入内容的处理
* 添加了格式化和解析函数参数后，对输入的处理
* 分了两个变量-inputValue和value，当聚焦时，使用inputValue，当失焦时，需对输入进行矫正，并改变value。value有存储的作用。
